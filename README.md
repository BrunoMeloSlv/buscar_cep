# 🚀 Preenchimento Automático de Endereços com Python

Este projeto automatiza o preenchimento de endereços a partir de CEPs utilizando a API gratuita do ViaCEP.

---

## 📌 Funcionalidades

- Leitura de planilha Excel com coluna de CEP
- Consulta automática na API ViaCEP
- Preenchimento de:
  - Logradouro
  - Bairro
  - Cidade
  - Estado
- Exportação para novo arquivo Excel
- Tratamento de erros (CEP inválido ou não encontrado)

---

## 🛠️ Tecnologias

- Python
- Pandas
- Requests
- Openpyxl
